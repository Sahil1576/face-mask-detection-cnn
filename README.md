# 😷 Face Mask Detection using CNN

A deep learning project that detects whether a person is wearing a face mask using a Convolutional Neural Network (CNN). The project includes a Streamlit web application for real-time image classification.

## 📌 Project Overview

This project performs binary image classification to detect whether a person is wearing a face mask.

### Classes
- 😷 Mask
- ❌ Non Mask
